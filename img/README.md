## Carpeta `img/`

Poné acá tus imágenes para productos y para la sección **Imágenes**.

### Cómo usar

- **Producto**: en `data.js` → `productsData`, usá:
  - `image: "img/archivo.jpg"`

- **Sección Imágenes**: en `data.js` → `imagesData`, usá:
  - `src: "img/archivo.jpg"`

### Recomendaciones

- Formatos: `.jpg` / `.png` / `.webp`
- Tamaño recomendado:
  - Productos: 800x800
  - Sección Imágenes (tipo banner): 1200x700
